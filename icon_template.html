How to save these as PNG files:

  1. Take a screenshot of each icon (use a tool that allows you to select an area)
  2. Crop to the exact dimensions (16x16, 48x48, 128x128)
  3. Save each image as PNG in the icons folder with the following names:
    • icon16.png
    • icon48.png
    • icon128.png
